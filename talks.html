Featured Presentations

From Passwords to Passkeys: The Evolving State of User Authentication

Minnebar 18 - Mainstage Presentation (2024)

Delivered to a packed auditorium at Minnesota's largest technology unconference, this talk explored the journey from traditional passwords to modern passwordless authentication methods.

Key Topics Covered:

  • History and problems with password-based authentication
  • WebAuthn and FIDO2 standards explained
  • Implementation strategies for passkeys
  • User experience considerations in authentication
  • Security implications and threat models
  • Migration paths for existing applications

Audience Impact:

Attendees left with practical knowledge to implement modern authentication in their applications, understanding both the technical implementation and user experience considerations that make passwordless authentication successful.
